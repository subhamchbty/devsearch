# DevSearch — Crawler

> Documentation crawler service powered by Express + TypeScript.

The crawler ingests official documentation sites, extracts main content, preserves heading hierarchy, separates code blocks, normalizes URLs, and deduplicates content.

## Setup

```bash
pnpm install
```

## Running

```bash
# Development (watch mode) — http://localhost:3001
pnpm dev

# Production
pnpm run build
pnpm start
```
