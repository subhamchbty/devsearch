# DevSearch — Embedder

> Embedding generation service powered by Express + TypeScript.

The embedder generates vector embeddings for indexed documentation chunks and queries, then stores them in the vector database for similarity search.

## Setup

```bash
pnpm install
```

## Running

```bash
# Development (watch mode) — http://localhost:3003
pnpm dev

# Production
pnpm run build
pnpm start
```
